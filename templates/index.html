3D BOUNDING BOX APPROXIMATION

USING DEEP LEARNING AND GEOMETRIC CONSTRAINTS

The goal of this project is 3D object detection and pose estimation from a single image. Comparing current techniques on 3D object detection that only regressese the 3D orientation of objects, our method for this project first regresses relatively stable 3D object properties using a deep convolutional neural network and combines the predicted properties with imposed geometric constraints provided by the corresponding 2D bounding box to produce the complete 3D bounding box.
